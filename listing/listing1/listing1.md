Что выведет программа? Объяснить вывод программы.
```go
package main
import (
    "fmt"
)
func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
	
}
```

Ответ:
```
Вывод программы: 77, 78, 79
Новый срез создается на базе массива а, с 1 по 4 элемент (не включая последний).
```
